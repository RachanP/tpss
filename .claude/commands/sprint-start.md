# Sprint Start Checklist

อ่าน `.claude/rules/sprint-status.md` แล้วรายงาน:

1. **Sprint ปัจจุบัน** — ชื่อ, วันที่, Module ที่เกี่ยวข้อง
2. **User Stories ใน scope** — แสดง ID และ SP ทั้งหมด
3. **Rules ที่ต้องโหลด** — อ่านไฟล์ rules ที่เกี่ยวข้องกับ Module นี้ แล้วสรุป constraints สำคัญ
4. **คำถามที่รอคำตอบ** — จาก "คำถามที่รอคำตอบจากลูกค้า" ถ้ามีที่เกี่ยวข้องกับ Sprint นี้
5. **DoD Checklist** — แสดง Definition of Done พร้อม checkbox

จบด้วยคำถาม: "พร้อมเริ่มงาน US อันไหนก่อน?"
